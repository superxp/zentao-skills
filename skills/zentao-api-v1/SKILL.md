# 禅道 API v1.0 Skill

## 技能描述

禅道项目管理系统 API v1.0 技能，支持通过 AI 调用禅道 REST API 进行项目管理操作。

## 功能列表

### 认证
- 获取访问令牌（Token）

### 产品管理
- 获取产品列表
- 获取产品详情
- 创建产品
- 获取产品的 Bug 列表
- 获取产品的需求列表

### 项目管理
- 获取项目列表
- 获取项目详情
- 创建项目
- 获取项目任务列表
- 获取项目成员列表

### 执行管理
- 获取执行列表
- 获取执行详情
- 创建执行

### 需求管理
- 获取需求列表
- 获取需求详情
- 创建需求
- 更新需求状态

### 任务管理
- 获取任务列表
- 获取任务详情
- 创建任务
- 更新任务状态
- 指派任务

### Bug 管理
- 获取 Bug 列表
- 获取 Bug 详情
- 创建 Bug
- 更新 Bug 状态
- 指派 Bug

### 测试用例管理
- 获取测试用例列表
- 获取测试用例详情
- 创建测试用例

### 用户管理
- 获取用户列表
- 获取当前用户信息

## 使用说明

1. 首先需要配置禅道服务器地址和登录凭据
2. 调用获取 Token 接口获取访问令牌
3. 后续所有 API 调用需在 Header 中携带 Token

## 配置项

| 配置项 | 说明 | 必填 |
|--------|------|------|
| zentao_url | 禅道服务器地址 | 是 |
| username | 登录用户名 | 是 |
| password | 登录密码 | 是 |

## 注意事项

> ⚠️ 说明：当前技能仍处于实验阶段。请谨慎使用，并**强烈建议**对 AI 发起的 API 调用进行人工审核。

- 本技能基于禅道 API v1.0 版本开发
- 需要禅道版本 >= 16.x
- API 调用需要相应权限
- 建议在测试环境中先行验证
